## 运行
```sh
yarn 
yarn dev
```

## 构建部署

```shell
yarn build
```

## 目的
加快浏览和下载 github 资源，解决 github 访问慢以及一些文件不能在线访问呢的问题

## 技术点
- vue3
- native-ui
- github api
